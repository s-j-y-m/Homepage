## ✨ 特性

- 🌙 明暗主题切换
- 🌸 樱花飘落效果
- 💭 一言语句显示
- 👁️ 界面隐藏(看壁纸)
- 📱 响应式设计
- 🎨 简约现代的界面风格

## 🛠️ 本地开发

```bash
# 克隆项目
git clone https://github.com/s-j-y-m/Homepage
cd Homepage

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```
